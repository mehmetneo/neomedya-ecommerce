# 🚀 E-ticaret Sitesi Oluşturucu

Bu sistem ile farklı müşteriler için e-ticaret siteleri hızlıca oluşturabilirsiniz.

## 📋 Özellikler

✅ **Tam E-ticaret Sistemi**
- Ürün yönetimi
- Kullanıcı kayıt/giriş
- Sipariş takibi
- Ödeme entegrasyonu
- E-posta bildirimleri
- Admin paneli

✅ **Hızlı Özelleştirme**
- Site adı değiştirme
- Renk şeması
- Domain ayarları
- Admin bilgileri

✅ **Otomatik Deployment**
- Vercel entegrasyonu
- Domain bağlama
- SSL sertifikası

## 🎯 Kullanım

### 1. **Yeni Site Oluşturma**

```bash
# Hızlı başlatma (interaktif)
npm run new-site

# Manuel özelleştirme
npm run customize "Site Adı" "domain.com" "admin@domain.com" "şifre123"
```

### 2. **Geliştirme**

```bash
# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev
```

### 3. **Production Deployment**

```bash
# Vercel'e deploy et
npm run deploy

# Manuel deployment
vercel --prod
```
